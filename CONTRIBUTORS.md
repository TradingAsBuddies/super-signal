## How to Get Added

If you have contributed code, documentation, testing, or ideas and would like to be
listed here, please:

- Open a pull request updating this file, or  
- Open an issue with the label `documentation` requesting to be added.

Please include:

- Your name (or preferred handle)
- A link to your GitHub profile
- (Optional) A short note about what you contributed
